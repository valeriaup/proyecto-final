# proyecto-final
Amigos de Cuatro.
Este proyecto consiste en el desarrollo de la estructura de un sitio web estático para una Organización No Gubernamental (ONG) ficticia, "Amigos de Cuatro", dedicada al rescate animal. El objetivo principal de este ejercicio académico fue demostrar la comprensión y aplicación de la sintaxis y la semántica de HTML5 sin recurrir a hojas de estilo (CSS) o scripts (JavaScript). El sitio simula una plataforma funcional con cinco páginas principales interconectadas, haciendo un uso riguroso de etiquetas semánticas (<header>, <nav>, <main>, <section>, <aside>, <footer>) y elementos complejos de formulario y multimedia, cumpliendo con los requisitos específicos del curso.

Inicio (index.html): Presenta la bienvenida, usa dos secciones de contenido (Propósito e Impacto) y destaca información complementaria en un elemento aside.

Acerca de (about.html): Estructura la información biográfica y de valores de la ONG, incluye una línea de tiempo (<ol>) y listas de cualidades clave.

Casos de Rescate (casos.html): Muestra una tabla con la lista de animales disponibles y detalla los tres programas de servicio principales de la ONG, usando la tabla y la estructura de secciones.

Galeria (galeria.html): Demuestra el uso de incrustaciones de medios: video y audio locales, videos de YouTube y mapas de Google Maps (mediante <iframe>), además de una galería de imágenes.

Contacto (contacto.html): Incluye un formulario completo con más de 10 tipos de campos diferentes y validación HTML básica.
